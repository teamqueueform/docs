## Step 1: Setup Campaign

1. Click **Start from scratch**.
2. You will be redirected to the customization dashboard.
3. Enter:
    - Campaign name
    - Start date (past, present, or future; time optional)
    - End date (optional)
4. If no end date is set, the campaign runs until manually closed.
    
5. Click **Next** to proceed.


![Campaign setup](image-placeholder)

## Step 2: Setup Reward
QueueForm offers three reward types:

#### 1. Milestone-Based Rewards
- Set referral milestones using plus/minus buttons or manual input.
- Assign a reward name for each milestone
    
    (e.g., “20 referrals → Amazon gift card worth ₹500”)
    
- Up to **5 milestones** can be added.
- Reward appearance can be customized later.

#### 2. Rank-Based Rewards (Waitlist)
- Define top ranks eligible for rewards
    
    (e.g., “Top 5 users receive Myntra voucher worth ₹2000”)
    
- Up to **5 rank milestones** can be added.
- Reward appearance is customizable later.

#### 3. Random Winner
- Define number of winners.
- Assign reward names.
- Up to **5 different rewards** can be configured.
- Customize reward appearance later.

Reward type can be changed anytimefrom the top selector.

## Step 3: Waitlist Questions
1. Three pre-made screens are available by default.
2. Preview screens using the up/down arrows.
3. Campaign preview modes:
    - Full screen
    - Popup view
    - Mobile view
    - Desktop view
4. Click **Edit Form** to modify questions.
### Form Editor Layout

#### Left Panel
- Short text
- Long text
- Email
- Phone number
- Date
- Star rating
- Address
- Country
- Website
- Multiple choice
- Opinion scale
- Date & time

#### Center Panel
Interactive preview with desktop and mobile views.

#### Right Panel
- Info: text and cover image
- Style: background, text, button colors (hex)

Each slide has its own info and style settings.

Fields can be marked as **required**.

Click **Save Changes** when done.

![Form editor](image-placeholder)


## Step 4: Customize & Integrate

#### 1. Customize

- Customize:
    - Welcome email
    - Verification email
    - Successful referral email
- Choose campaign placement:
    - Email widget (available)
    - Single-page form (in production)
    - Landing page (in production)

Click **Save Changes** to continue.

#### 2. Integrate
QueueForm provides embed code for:

- Floating email widget
- Embedded email
- Embedded form

Integration guides are available for:

- WordPress
- Framer
- Webflow
- Shopify
- React
- HTML custom code

You can also connect:

- Webhooks
- Bitrix24
- Beehiiv
- Mailchimp
- Meta Pixel
- Google Analytics

![Embed options](image-placeholder)


## Step 5: Go Live
1. Click **Done** to publish
2. Copy the campaign link
3. Track live submissions and referrals through **dashboard**

#### Campaign Actions (via 3-dot menu)
- View responses
- Share QueueForm
- Edit form
- Duplicate form
- Pause or delete campaign
